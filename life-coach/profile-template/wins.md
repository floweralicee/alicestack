## How to use this file

The skill reads this to mirror your actual capability back to you — not as praise,
but as pattern recognition. "Here's what you did. Here's what it proves. Here's the
gap between this evidence and how you're currently describing yourself."

Don't filter. Log the small ones. "Sent 5 emails" counts. "Showed up anyway" counts.
The pattern only becomes visible when the data is complete.
