# Patterns
> Loops you've caught yourself running. Named so you can see them coming.

The skill uses this file to create your Villagers — the named pattern characters
that appear when you're running a loop. The more honestly you write here,
the more precisely the skill can call it.

## How to write a good pattern entry

Bad: "I procrastinate"
Good: "When I have something important to ship, I spend the first 2 hours reorganizing
my workspace or building a better system for the task instead of doing the task. It
shows up most when I'm afraid the output won't be good enough."

The bad version gives the skill nothing. The good version becomes a Villager that
can walk in the room and say something specific.
